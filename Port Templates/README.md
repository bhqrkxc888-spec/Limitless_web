## ğŸš€ Quick Start: Create a Port Guide

### Method 1: Single Port (10 minutes)
1. Open `PORT_TEMPLATE_FOR_AI.md`
2. Copy entire content
3. Go to Claude or Perplexity
4. Say: "Fill out this cruise port guide for **[PORT NAME]**"
5. Paste the template
6. AI generates complete guide
7. Save as `port-slug.md` (e.g., `barcelona.md`)
8. Upload via **Admin > Port Management > Upload Markdown**

### Method 2: Batch (Multiple Ports)
1. Open `AI_PROMPT_FOR_PORTS.md`
2. Follow "Multi-Port Batch" instructions
3. Generate 3-5 ports at once
4. Save each separately
5. Upload all to admin
