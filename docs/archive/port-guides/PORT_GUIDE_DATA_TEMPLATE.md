## Notes for Data Collection

- **NO PRICE SYMBOLS** - Do not include £, ££, €, $ or any pricing
- **NO EMOJIS** - Keep all text clean and professional
- **NO EM-DASHES** - Use "to" instead (e.g., "2 to 3 hours" not "2–3 hours")
- **BRITISH ENGLISH** - Use British spellings (e.g., "centre" not "center")
- **FACTUAL TONE** - Keep it informative and helpful, not sales-y
- **6 ATTRACTIONS MAX** - We show 6 in a 3x2 grid, no more
- **4 RESTAURANTS MAX** - We show 4 in a row
